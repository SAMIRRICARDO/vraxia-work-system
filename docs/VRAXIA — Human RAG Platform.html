Pipeline Human RAG

Captura → Destilação → Perfil Psicológico → Gêmeo Cognitivo → Resposta Calibrada

Fase 1 — Captura LLaMA 3.1 8B · LOCAL

Entrevistas conversacionais com o profissional. O agente extrai padrões de raciocínio, heurísticas e exceções — não apenas fatos. Transcrições de áudio/vídeo via Whisper local.

Input: conversa natural → Output: DecisionPattern[] com confidence scores
Fase 2 — Destilação text-embedding-3-small

Padrões extraídos são convertidos em CognitiveChunks com dois embeddings independentes: semântico (conteúdo) e contextual (domínio + conteúdo). Armazenados em pgvector com índice IVFFlat.

Input: DecisionPattern[] → Output: CognitiveChunk[] indexados em pgvector
Fase 3 — Avaliação Psicológica PsychAgent · LLaMA 3.1 70B

20 trocas conversacionais. 6 instrumentos aplicados simultaneamente de forma invisível. Resultado: PsychologicalProfile JSON que calibra TODOS os aspectos da geração.

Input: sessão 20 trocas → Output: PsychologicalProfile com BiasAlerts
Fase 4 — Retrieval + Geração Calibrada GPT-4o-mini · CHEAP MODE

Query vetorizada recupera Top-K chunks por similaridade coseno. buildCalibratedPrompt injeta Big Five, vieses (medium+high), estilo decisório e padrões cognitivos. Resposta gerada como o profissional responderia — incluindo limitações reais.

Input: query + profile + chunks → Output: resposta calibrada com biasesActivated[]
