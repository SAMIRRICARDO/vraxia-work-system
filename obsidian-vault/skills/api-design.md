# API Design Skill

## Goal
Criar APIs escaláveis e econômicas.

## Constraints
- minimizar chamadas LLM
- usar cache
- usar DTOs
- validar tudo com Zod

## Flow
1. validar schema
2. criar service
3. criar repository
4. criar testes
5. documentar

## Standards
- repository pattern
- service layer
- sem lógica em controllers
